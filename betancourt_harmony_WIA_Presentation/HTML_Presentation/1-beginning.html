Tilted Page Scroll

A plug-in by Pete R

Covered by Harmony Betancourt for WIA

What is it?

  • Located on Github
    https://github.com/peachananr/tiltedpage_scroll
  • Uses
    • Portfolios
    • Image Galleries

Page Structure

Head


	<!-- jQuery and Plugin calls -->
		<script src="http://code.jquery.com/jquery-1.9.1.js"></script>
		<script type="text/javascript" src="js/jquery.tiltedpage-scroll.js"></script>
		<!-- CSS call for plugin -->
		<link href='css/tiltedpage-scroll.css' rel='stylesheet' type='text/css'>
		<!-- CSS Stylesheet changes by me -->
		<link href='css/style.css' rel='stylesheet' type='text/css'>

HTML


<div class="main"> 
	<section class="page1"> 
	   <div class="page_container">
	   <!-- First page content -->
	   </div>
		</section> 
		<section class="page2">
	   	<div class="page_container">
	   <!-- Second page content -->
	   </div> 
		</section> 
		<section class="last-page"> 
	   	<div class="page_container">
	   <!-- Last page content -->
	   </div>
		</section> 
</div>

jQuery


$(".main").tiltedpage_scroll({
	sectionContainer: "> section",
	angle: 50,
	opacity: true,
	scale: true,
	outAnimation: true
});

CSS


.tps-section:nth-child(1n+1) .tps-wrapper {
	background: url("../images/img1.jpg") center center;
	background-size: cover;
}
